Readme WebStringFormater

What is it ?

This program allow you to convert non-html characters, from the clipboard, a file or by writing them inside the input zone. Conversion will be displayed into the output zone. You can also convert text inside your clipboard without flushing it.

Demo

The following supported characters are note well encoded. Try it yourself !

Unsupported characters

Due to programming language use, following characters are not supported by the program.

&

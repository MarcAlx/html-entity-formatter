Readme WebStringFormater

A quoi ça sert et comment ça marche ?

Ce programme permet de convertir automatiquement les caractères non-HTML d'un fichier (via le bouton Format a file), du presse-papier de votre ordinateur (bouton Clipboard -> Format) ou encore à partir de la zone de texte supérieure en caractères HTML

Le texte résultant de cette conversion est alors remplacé (selon la conversion choisie) directement dans le fichier ou alors dans la zone de texte inférieur. Il peut même être auto copier dans le presse-papier (bouton Format -> Clipboard)

Vous voulez formater rapidement une chaine, copier la et cliquez sur le bouton : Clipboard -> Format -> Clipboard, la chaine sera automatiquement formatée dans votre presse-papier, vous n'avez plus qu'à la coller.

Demo

Les caractères ci-desous ne sont pas codés en HTML (vous pouvez vérifer le code source du fichier). Lancez le programme cliquez sur : format a file, observez le code source de ce document HTML. Ou bien faites des copier-coller à partir de cette liste directement dans la zone de texte supérieure du programme et appuyez sur Format ou Format -> Clipboard.
